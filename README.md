## jswooddesign.nl

**Click-Designs**
Release Date: 10/02/2021
Author: Mark Hofmeijer (m.hofmeijer@click-designs.nl)

## Tech Stack

- Front-end: [React](https://reactjs.org/), [TypeScript](https://www.typescriptlang.org/)
- Web Framework: [Next.js](https://nextjs.org/)
- CSS Framework: [Tailwind CSS](https://tailwindcss.com/)
- Hosting: [Vercel Platform](https://vercel.com/)
- Domain Registrar: [Namecheap](https://www.namecheap.com/)
- Headless CMS: [Contentful](https://www.contentful.com/)
- Form Submissions: [Formspree](https://formspree.io/)

## Local Development

First, clone the repository. Then run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
